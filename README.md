lua-requests
====

The same friendly python Requests interface for Lua!

Under construction... Stay tuned.


Tests
====

Tests are written using [busted](http://olivinelabs.com/busted/ "Busted home page").

Install `busted`:

	$ luarocks install busted

Run Tests:

	$ busted -p _tests tests
